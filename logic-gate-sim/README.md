# Logic Gate Simulator
---
A simple logic-gate simulator that shows the main idea about logic gates, how it works in various inputs.
It includes AND, NAND, OR, XOR, and NOT. Well it's just a project from a professor's recommendation. Just wants me to 
create a visual representation/simulation of the logic gate for his students.
I will use SDL here because I don't think I have to use OpenGL/Vulkan for something so simple.
